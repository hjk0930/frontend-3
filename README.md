## 🎯 1. 기획 배경 및 해결 과제
* **현재 상황 및 문제점:** 학과 내 디스코드에서 롤 내전을 하는 경우가 있습니다 그럴때마다 팀을 정해야하는데 팀을 정하는데 너무 많은 시간이 소요가 되는 경우가 있어 불편한 경우가 있었습니다
* **우리가 제공할 해결책:** 내전을 하는 사람들의 랭킹 티어를 입력하면 빠르게 밸런스 있는 팀이 만들어집니다

## 👤 2. 타겟 유저 페르소나
* **인구통계학적 특성:** 학교 디스코드방 내에서 인원을 차출해 자주 내전을 진행하며 팀 구성에 어려음을 겪는 LOL 플레이어들
* **불편함:** 게임을 하기 위해 모여서 정작 팀을 짜는 데에 더 많은 시간을 들이게 되는 것
* **기대하는 것:** 포지션이나 챔프의 특성등을 스스로 고려할 필요 없이 팀을 자동으로 빠르게 정해주는 것

## ⭐ 3. 핵심 기능
> 프론트엔드 환경에서 3주 내에 구현할 수 있는 핵심적인 기능 딱 3가지를 정의합니다.
1. **[선수 프로필 입력 기능]**: 네임, 티어, 라인, 주 챔피언 등 내전 팀 구성에 필요한 기본 정보를 입력받는다.
2. **[스탯 카드 생성 기능]**: 입력된 정보를 바탕으로 선수별 카드 UI를 생성하여 한눈에 비교할 수 있도록 한다.
3. **[팀 편성 기능]**: 계산표에 따라 선수 점수를 산출하고, 팀 평균 점수 차이가 최소화되도록 내전표를 생성한다.

## 🗺️ 4. 화면 설계 및 유저 플로우
* 사용자가 우리 웹사이트에 접속하여 목표를 달성하기까지의 클릭 흐름을 단계별로 작성합니다.
  1. `메인 페이지` 접속
  2. 선수 기본 정보 입력 (각 선수의 닉네임, 티어, 주 라인, 주 챔피언을 입력.)
  3. 선수 카드 생성 (입력된 정보를 카드 형태로 화면에 표시.)
  4. 선수 목록 확인 및 수정
  5. 10명 등록 완료 후 팀 생성
  6. 자동 팀 밸런싱 진행 (입력된 티어와 라인을 기준으로 각 선수의 점수를 계산하고, 양 팀의 평균 점수가 최대한 비슷해지도록 팀을 나눔)
  7. 내전표 결과 확인 (내전표에서 각 팀의 선수 구성, 라인, 챔피언, 개인 티어, 팀 평균 점수 차이 등을 확인함)
